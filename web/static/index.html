New Pathways Per Hour

The slider controls the number of pathways started per hour. Each pathway contains multiple events which generate messages.
You can also control the pathways started per hour with the following command:
curl -XPOST THIS-PAGE-ADDRESS/pathwayRate -d 'value=200'

Run A Pathway or Send A Message
You can enter the following:
  • A HL7v2 message with each segment in a new line, or
  • The name of a pathway such as aki_scenario_1 or
  • The name of a pathway plus the MRN of a patient, for instance aki_scenario_1: 1234567890 or
  • The name of a pathway plus the first and last name of a patient, for instance aki_scenario_1: Tina Turner or
  • The full pathway in the yml format.
See the documentation for more information on this text box and how to write pathways.
